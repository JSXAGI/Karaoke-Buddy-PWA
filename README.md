# Karaoke Buddy PWA

カラオケ好きなユーザー向けに作成した、PWA対応のWebアプリです。

## アプリ概要

`Karaoke Buddy PWA` は、以下を1つにまとめたアプリです。

- 歌いたい曲の登録
- 曲ごとのメモ管理
- 採点結果の保存
- 平均点・最高点の可視化
- お気に入り管理
- ランダムセットリスト生成
- オフライン利用
- JSONバックアップ / 復元

ブラウザから開いてホーム画面へ追加することで、アプリのように利用できます。

## 主な機能

### 1. 曲管理
- 曲名
- アーティスト
- キー調整
- 難易度
- お気に入り
- メモ

を登録できます。

### 2. 採点履歴
各曲に対して採点結果を登録できます。

- 点数: `0` 〜 `100`
- 小数点入力対応
- コメント保存対応

### 3. ダッシュボード
以下を自動集計します。

- 登録曲数
- 平均点
- 最高点
- お気に入り数

### 4. セットリスト生成
以下のモードでセットリストを作成できます。

- 完全ランダム
- お気に入り優先
- 低得点曲優先

### 5. オフライン対応
`ServiceWorker.js` により、初回アクセス後は基本的なアプリシェルをキャッシュし、オフラインでも利用できます。

### 6. バックアップ / 復元
データをJSON形式でエクスポートし、インポートできます。
